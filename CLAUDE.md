This is the Audit Trail Q&A System — an AI-assisted audit support tool for internal finance teams.
Full spec: docs/design.docx
Phased build plan: docs/roadmap.docx

Stack: FastAPI + SQLAlchemy + PostgreSQL backend, React + TypeScript + Tailwind frontend,
Qdrant (or ChromaDB) vector DB, Claude API for generation, MCP Python SDK for tool calling.
